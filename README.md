##  Project Overview
This project implements **ECG (Electrocardiogram) peak detection** using signal processing techniques.  
The main objective is to identify **R-peaks**, which are essential for heart rate analysis and cardiovascular assessment.

The implementation is provided in a Jupyter Notebook:

> `ECG peak detection.ipynb`

---

##  Objectives
- Preprocess ECG signals to remove noise  
- Detect R-peaks accurately  
- Visualize ECG signals and detected peaks  
- Analyze signal characteristics
